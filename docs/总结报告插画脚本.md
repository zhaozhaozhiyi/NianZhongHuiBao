# 2025年度总结报告插画脚本

## 封面插画

**场景描述：**
- 画面中央是一位产品经理站在交叉路口，左手拿着一叠"题目"（代表做题的人），右手伸向远方，指向一个发光的"系统架构图"（代表出题的人）
- 背景分为两部分：左侧是传统的工作场景（加班、重复性任务），色调偏灰暗；右侧是创新的系统化场景（自动化、高效协作），色调明亮温暖
- 时间标识：2025年
- 整体氛围：从执行者到设计者的转变，充满希望和突破感

**色彩方案：**
- 主色调：深蓝色（专业）渐变到金色（成就）
- 辅助色：灰色（过去）到橙色（未来）

## 插画设计说明

### 整体风格
- **风格定位**：现代商务插画风格，专业但不失温暖
- **人物形象**：简约、专业，突出团队协作
- **色彩体系**：蓝色系（专业）+ 金色（成就）+ 暖色（团队）

### 视觉元素
- **图标系统**：目标、协作、系统、数据流、时间线
- **动态元素**：箭头、连接线、数据流、能量线
- **场景设置**：办公室、会议室、跨地区、项目现场

### 构图原则
- **中心聚焦**：重要内容在中心
- **对比构图**：问题 vs 解决，过去 vs 未来
- **动态感**：用线条、箭头表现行动和变化

### 色彩心理学
- **蓝色**：专业、信任、稳定
- **金色**：成就、价值、突破
- **橙色**：活力、团队、温暖
- **红色**：挑战、警示（谨慎使用）
- **绿色**：成功、成长、突破
